Create Diploma

As a University, creating authenticate Certificates and presenting them to students is crucial. ValidateIt allows Universities to enter Student details and uses a Smart Contract deployed using Truffle to Create Certificates. When an authorized user clicks on "Create Certificate", the Smart Contract concatenates all given values and generates a hash. This hash is mapped to a boolean value True and added to Online Blockchain. This record is now ready to be Verified by anyone at any time.
